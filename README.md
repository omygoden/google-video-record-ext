# AV Recorder Pro (Chrome Extension)

## 功能
- 视频录制：导出 MP4
- 音频录制：导出 MP3 或 WAV
- 默认音频格式：WAV
- 点击扩展图标会打开独立控制窗口（不会因点击浏览器其他区域自动关闭）
- 内置标签页音频增益（2x）以减轻导出音量偏小问题

## 安装
1. 打开 Chrome，访问 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录：`F:\gowork\google-ext\collect-video-ext`

## 使用
1. 点击扩展图标打开面板
2. 在“视频录制”中开始/停止并保存 MP4
3. 在“音频录制”中选择 MP3 或 WAV 后开始/停止保存

## 注意
- 录制来源是你点击扩展图标时所在的标签页，请先切到目标页面再点图标。
- MP4/MP3 依赖浏览器编码器支持；若你的环境不支持，插件会提示。
- WAV 导出由插件内置转码完成。
